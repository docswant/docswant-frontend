> ## 개발 인원 및 기간

- 개발 기간: 2022.03 ~ 2022.06

  <br/>

> ## 💻 Tech Stack

- ### Skills

  React, Styled-components, Redux

- ### ETC
  JWT, ESL(전자가격표시기)

<br/>

> ## 💨 구현 기능

- 로그인/회원가입
- 마이페이지
- 의사
  - 환자 정보 CRUD
  - 회진 전 질문 CRUD
  - 회진 일정 CRUD
- 환자
  - 제공 받은 질문 답변/수정/삭제
  - 건의사항 CRUD
- ESL
  - 환자 로그인 시 환자 ESL에 사용자 정보 표기
  - 의사가 회진 일정 시 환자 ESL에 회진 일정 및 회진 순서 표기
